### Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Start the app:**

    ```bash
    npx expo start
    ```

3.  **Run on device/emulator:**
    - Press `a` for Android Emulator.
    - Press `i` for iOS Simulator.
    - Scan the QR code with the **Expo Go** app on your physical device.
